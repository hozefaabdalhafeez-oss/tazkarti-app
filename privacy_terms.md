# شروط الاستخدام وسياسة الخصوصية لتذكرتي

شروط الاستخدام:

- استخدام التطبيق لحجز التذاكر فقط
- إدخال بيانات صحيحة عند الحجز
- الالتزام بسياسات شركات النقل

سياسة الخصوصية:

- حماية بيانات المستخدمين
- عدم مشاركة البيانات بدون إذن
- تأمين معلومات الحجز والدفع

حقوق المستخدم:

- تعديل بياناته
- مراجعة حجوزاته
- طلب المساعدة
